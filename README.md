# Student Management System (Java)

A console-based Java application designed to manage student records efficiently.  
The system allows users to perform CRUD operations, analyze student performance, and manage academic data using Object-Oriented Programming principles.

---

## 🚀 Features

- Add a new student
- Remove student by ID
- Update student information
- Search student by ID
- List all students
- Sort students by GPA
- Calculate average GPA
- Display student performance summary

---

## 🛠 Technologies Used

- Java
- Object-Oriented Programming (OOP)
- Java Collections Framework
- File Handling
- CSV Data Storage
